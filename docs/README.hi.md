# grit

**AI एजेंट्स के लिए Git — शून्य मर्ज कॉन्फ्लिक्ट, कितने भी पैरेलल एजेंट, एक ही कोडबेस।**

> जब 50 एजेंट एक ही रिपो पर काम करते हैं, तो git टूट जाता है। Grit नहीं टूटता।

## समस्या

एक कोडबेस पर N AI एजेंट्स को समानांतर चलाने से O(N²) मर्ज कॉन्फ्लिक्ट बनते हैं। Git **लाइन लेवल** पर काम करता है — जब दो एजेंट एक ही फाइल में अलग-अलग फंक्शन एडिट करते हैं, तो Git कॉन्फ्लिक्टिंग हंक देखता है और मर्ज फेल हो जाता है।

## समाधान

Grit tree-sitter का उपयोग करके **AST/फंक्शन लेवल** पर लॉक करता है। हर एजेंट एडिट करने से पहले विशिष्ट फंक्शन रिज़र्व करता है। एक ही फाइल में अलग-अलग फंक्शन कभी कॉन्फ्लिक्ट नहीं करते। एजेंट आइसोलेटेड git worktree में काम करते हैं और मर्ज ऑटोमैटिकली सीरियलाइज़ होते हैं।

```bash

```

## क्विक स्टार्ट

```bash
cd your-project
grit init                    # AST पार्स करें, सिंबल इंडेक्स बनाएं

# एजेंट एडिट करने से पहले फंक्शन रिज़र्व करता है
grit claim -a agent-1 -i "वैलिडेशन जोड़ें" \
  src/auth.ts::validateToken \
  src/auth.ts::refreshToken

# एजेंट आइसोलेटेड worktree में काम करता है: .grit/worktrees/agent-1/
# ... फाइलें एडिट करें ...

# पूरा: ऑटो-कमिट, रीबेस, मर्ज, लॉक रिलीज़
grit done -a agent-1
```

## सेशन वर्कफ़्लो (GitHub इंटीग्रेशन)

```bash
grit session start auth-refactor        # ब्रांच grit/auth-refactor बनाएं
# ... एजेंट claim, काम, done ...
grit session pr                         # ब्रांच push + GitHub PR बनाएं
grit session end                        # लॉक क्लीनअप, बेस ब्रांच पर वापस
```

## कमांड

```
grit init                                    # सिंबल इंडेक्स इनिशियलाइज़ करें
grit claim -a <agent> -i <intent> <syms...>  # सिंबल लॉक + worktree बनाएं
grit done  -a <agent>                        # मर्ज + लॉक रिलीज़
grit status                                  # एक्टिव लॉक दिखाएं
grit symbols [--file <pattern>]              # इंडेक्स किए सिंबल की सूची
grit plan <symbols...>                       # उपलब्धता जांचें (dry-run)
grit release -a <agent> <symbols...>         # विशिष्ट लॉक रिलीज़ करें
grit gc                                      # एक्सपायर्ड लॉक क्लीन करें
grit heartbeat -a <agent>                    # लॉक TTL रिफ्रेश करें
grit watch                                   # रियल-टाइम इवेंट स्ट्रीम
grit session start|status|pr|end             # फीचर ब्रांच लाइफसाइकल
grit config set-s3|set-local|show            # बैकएंड कॉन्फिगरेशन
```
